Wizard

Default

The Wizard variant of Accordion forces the user to move through the sections in order. Each yeti-accordion-section's status, is-openable, is-open, and heading text properties can be programmatically set. The parent yeti-accordion component will also set and maintain some properties. See the property declarations for more information.

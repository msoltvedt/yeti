YetiTableContents Interface

The Yeti Table component expects you to provide data via the contents attribute. That data should match the following format: 


/******************************************** A TypeScripty/pseudo-code interface definition ********************************************/

YetiTableContents = {

    head? : {
      cssClass?: string,
      id?: string,
      rows : YetiTableRow[]
    },
  
    body : {
      cssClass?: string,
      id?: string,
      rows : YetiTableRow[]
    },
  
    foot? : {
      cssClass?: string,
      id?: string,
      rows : YetiTableRow[]
    },

}
  

YetiTableRow = {
    cssClass?: string,
    isExpandable?: boolean,
    id?: string,
    cells: YetiTableCell[]
}
  
  
YetiTableCell = {
    cssClass?: string,
    isHeading?: boolean,
    id?: string,
    value: string
}

Technical Notes

  • Yeti Table only supports JSON for now.

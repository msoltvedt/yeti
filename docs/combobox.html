Selection Types

Combobox has three different selection types: manual (default), automatic, and inline. The selection type describes how the user makes their selection from the list. See the examples below for an explanation of each selection type.

Manual (Default)

The manual selection type is the default, and it requires the user to manually select an option from the list by clicking on it or navigating to it with the keyboard and hitting enter.

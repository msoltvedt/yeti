Technical Notes

  • Yeti Modal will trap focus while open.
  • The actual modal will be appended to the end of the body tag, and all other contents of the body will be wrapped in a div so we can prevent scrolling while the modal is open.
  • Upon opening, the Modal will assign focus the first focusable thing in its content area. If nothing in the content area is focusable, then the content area itself will gain focus.
  • Upon closing, the Modal will return focus to whatever had it before it opened.

Alignment
Main Axis

The main axis for rows is horizontal, whereas for columns it's vertical. Use the .yeti-layout-main_axis-... classes to adjust alignment relative to that axis. Possible options include: -center, -start, -end, and -stretch.

Size, Width, and Height

Yeti uses a base unit system, in which sizes are specified as multiples of a common base unit (currently 0.25rem, or about 4px for most users). You can quickly specify widths, heights, and sizes as multiples of this unit.

Colors
